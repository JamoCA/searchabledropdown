A jQuery plugin which extends normal dropdown (select) elements to be searchable.

Project homepage: [https://sourceforge.net/projects/jsearchdropdown/](https://sourceforge.net/projects/jsearchdropdown/)

The drop down below contains more than 1000 entries. Try searching.

## Features:

- Tested in IE 7.x, IE 8.x, Firefox 3.5.x, Safari 4.x, Opera 10.x, Chrome 3.x
- Searching in select boxes without losing normal select box behaviors
- Support modification of select object at runtime
- Use keys to navigate through the list of results
- Use key ESCAPE to abort your searching
- NOTE: There is no support for IE 6.x or lower IE versions

## Usage:

### Simple Setup

```javascript
$(document).ready(function() {
	$("select").searchable();
});
```

```javascript
$(document).ready(function() {
	$("select").searchable({
		maxListSize: 100,                       // if list size are less than maxListSize, show them all
		maxMultiMatch: 50,                      // how many matching entries should be displayed
		exactMatch: false,                      // Exact matching on search
		wildcards: true,                        // Support for wildcard characters (*, ?)
		ignoreCase: true,                       // Ignore case sensitivity
		latency: 200,                           // how many millis to wait until starting search
		warnMultiMatch: 'top {0} matches ...',  // string to append to a list of entries cut short by maxMultiMatch
		warnNoMatch: 'no matches ...',          // string to show in the list when no entries match
		zIndex: 'auto'                          // zIndex for elements generated by this plugin
	});
});
```
